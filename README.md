# Restaurante Apolo 

Projeto desenvolvido como parte do processo seletivo da **EJECT**. O objetivo foi criar uma plataforma web moderna e responsiva para um restaurante italiano de alto padrão, focando em usabilidade, design elegante e funcionalidade.

## Tecnologias Utilizadas

* **HTML5:** Estrutura semântica e acessível.
* **CSS3:** Estilização avançada com uso de:
    * **Flexbox & Grid:** Para layouts dinâmicos e alinhamentos complexos.
    * **Variáveis CSS:** Para padronização de cores e temas.
    * **Medidas Relativas:** Garantindo escalabilidade e acessibilidade.
    * **Media Queries:** Responsividade.
* **JavaScript:** Lógica para manipulação de DOM e controle do Modal.

## Funcionalidades

* **Página Inicial:** Home interativo e seções de destaques.
* **Sobre Nós:** História do restaurante com layout intercalado responsivo.
* **Blog:** Grid de artigos sobre culinária italiana.
* **Sistema de Reserva:** Modal funcional integrado ao Header em todas as páginas.
* **Responsividade:** Interface adaptável para celulares, tablets e desktops.

## Como executar o projeto

1. Clone o repositório:
   ```bash
   git clone https://github.com/marcelo0404/apollo-restaurant-system

2. Navegue até a pasta do projeto:
   cd restaurante-apolo

3. Abra o arquivo *index.html* em seu navegador ou utilize a extensão Live Server no VS Code.
